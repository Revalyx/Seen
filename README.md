# Seen
Repositorio de la web de Seen, una "cámara de conocimiento" que permite registrar todo el contenido audiovisual consumido. En un futuro, la aplicación tomará un rumbo más social, siendo pensado para existir perfiles de usuario que pueden seguir e incluso comentar las publicaciones de otros usuarios.

La app está pensada para ser un ecosistema completo, comprendiendo así tanto el entorno web, como la aplicación Android y la aplicación de escritorio, permitiendo accesibilidad desde cualquier dispositivo y en cualquier momento
